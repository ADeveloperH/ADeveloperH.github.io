### 2 使用步骤 ###

### 2.1 修改项目下gradle.properties文件 ###

添加如下命令：

```java
android.useDeprecatedNdk=true
```

### 2.1 配置app下的build.gradle ###

defaultConfig下增加ndk配置：

```java
defaultConfig {
        applicationId "com.mobile.androidtest"
        minSdkVersion 11
        targetSdkVersion 23
        versionCode 1
        versionName "1.0"
        
        ndk{
            //生成.so的文件名称
            moduleName "hellojnilib"
            abiFilters "armeabi"
        }
    }
```
